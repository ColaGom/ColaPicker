ColaPicker
=====
ColaPicker is so simple library for multiple image picker from your android device.
it's with glide. so fast , simple :)


Import
=====
```gradle
repositories {
  ...
  maven { url "https://jitpack.io" }
}

dependencies {
  ...
  compile 'com.github.colabearstudio:colapicker:1.0.2'
}
```
